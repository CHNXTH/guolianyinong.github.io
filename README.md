# 果链益农官方网站

这是果链益农的官方网站项目，展示了我们的品牌使命、产品信息和创新模式。

## 项目结构

```
.
├── index.html          # 首页
├── css/
│   └── style.css      # 主样式文件
├── js/
│   └── main.js        # 主JavaScript文件
├── assets/            # 媒体资源文件夹
│   ├── hero-video.mp4 # 首页背景视频
│   └── hero-bg.jpg    # 背景图片（视频加载失败时的备用）
└── README.md          # 项目说明文档
```

## 功能特点

- 响应式设计，适配各种设备尺寸
- 现代化的UI/UX设计
- 流畅的动画效果
- 优化的性能表现
- 无障碍访问支持

## 技术栈

- HTML5
- CSS3 (Flexbox, Grid, 动画)
- JavaScript (ES6+)
- Font Awesome 图标库
